1.kirim variabel session username dari controller ke view
  - /controllers/game.php

    function play(){
        $data['username'] = $_SESSION['USER']['USERNAME'];
        $this->load->view('game',$data);
    }

  - /views/game.php
   <param name="FlashVars" value="username=<?= $username ?>"/> (sudah)

2.FlashVars adalah variabel (url query string) yang bisa ditangkap oleh as3
  - https://helpx.adobe.com/flash/kb/pass-variables-swfs-flashvars.html
  - http://stackoverflow.com/questions/4671867/how-do-i-access-flashvars-in-as3-and-add-them-to-an-existing-text-field
  kita dapat mengirim variabel sebanyak mungkin (value="username=<?= $username ?>&var2=value2&var3=value3&x=a&y=b"). (sudah)

Security:
Karena ada pengecekan session di /controllers/game.php maka user yang belum login tidak akan bisa mengakses langsung halaman :
http://localhost/vidyanusa/index.php/game/play. (sudah)

konfigurasi memanggil swf lain (sudah) :
 <param name="base" value="<?php echo $this->config->item('swf') ?>/"/>
